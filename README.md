# 🏏 AstaarBets

A modern 3D IPL prediction and sports analytics web application built for cricket fans.  
The platform provides match insights, toss predictions, team statistics, and interactive visualizations for IPL 2026 teams.

> ⚠️ This project is developed for educational and entertainment purposes only.

---

## 🚀 Features

- 🎯 IPL Match Winner Predictions
- 🪙 Toss Winner Predictions
- 📊 Team Performance Analytics
- 🌐 Modern 3D UI Design
- ⚡ Real-time Match Interface
- 📱 Responsive Design
- 🔥 Interactive Animations
- 🏆 IPL 2026 Winner Prediction Section

## 🛠️ Tech Stack

- Frontend: React.js / Next.js
- Styling: Tailwind CSS
- 3D Graphics: Three.js
- Backend: Node.js + Express
- Database: MongoDB
- Authentication: JWT

## Prompt
Create a modern, responsive 3D website for IPL (Indian Premier League) match predictions and betting simulation.

Tech Stack:

* Frontend: HTML, CSS, JavaScript
* Use Three.js for 3D effects
* Optional: React.js
* Use localStorage for saving user bets

IPL Teams (IMPORTANT – use ONLY these 10 teams):

* PBKS (Punjab Kings)
* RCB (Royal Challengers Bangalore)
* MI (Mumbai Indians)
* CSK (Chennai Super Kings)
* LSG (Lucknow Super Giants)
* GT (Gujarat Titans)
* SRH (Sunrisers Hyderabad)
* DC (Delhi Capitals)
* RR (Rajasthan Royals)
* KKR (Kolkata Knight Riders)

Main Features:

1. 3D Homepage:

* Animated cricket stadium using Three.js
* Navbar: Home, Matches, Predictions, Bet, IPL 2026 Winner, Leaderboard
* Smooth transitions and hover effects

2. Teams Section:

* Display all 10 IPL teams as 3D cards
* Each card shows logo, name, and stats
* Add hover tilt and glow effects

3. Match Prediction:

* Show upcoming matches (use sample IPL 2026 data)
* Each match includes:

  * Team vs Team
  * Predict Match Winner
  * Predict Toss Winner
* Add animated probability bars

4. Betting Simulation:

* Virtual coins system (no real money)
* User can:

  * Bet on match winner
  * Bet on toss winner
* Show balance, win/loss result
* Store data using localStorage

5. IPL 2026 Winner Section:

* Show all 10 teams
* User selects one team as tournament winner
* Display odds (static values)
* Save selection

6. Leaderboard:

* Rank users based on virtual coins
* Show top players (use dummy data if no backend)

7. UI/UX:

* Dark neon theme with glassmorphism
* 3D hover animations
* Smooth scrolling
* Mobile responsive

8. Extra:

* Sound effects for bets
* Notifications for results
* Loading animations

Important:

* This is a betting simulation using virtual currency only
* Do NOT include real money or payment integration

Output:

* Clean, modular code
* Separate HTML, CSS, JS files
* Ready-to-run project
